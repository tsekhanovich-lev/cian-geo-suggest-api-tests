# CIAN API Testing

## Описание

Проведено ручное тестирование API эндпоинта:

GET https://api.cian.ru/geo-suggest/v2/suggest/

Эндпоинт предназначен для получения гео-подсказок (города, районы) по введённому тексту.

---

## Параметры запроса

- offerType — тип недвижимости (flat)
- query — поисковый запрос
- regionId — ID региона (2)
- dealType — тип сделки (rent)
- source — источник (serp)

---

## Проведённые проверки

### Валидный запрос
- query = "сaнкт-петербург"  
Результат: 200 OK, корректный JSON с подсказками

### Невалидный (несуществующий) запрос
- query = "вфываыва"  
Результат: 200 OK, пустой список

### Негативный сценарий
- query = "!@#$"  
Результат: 500 Internal Server Error

---

## Найденные дефекты

### BUG-001: 500 при некорректных символах

Шаги:
1. Отправить GET-запрос
2. Указать query = "!@#$%^&*"

Фактический результат:
- 500 Internal Server Error

Ожидаемый результат:
- 200 с пустым результатом или 400 с описанием ошибки

Серьёзность: High
